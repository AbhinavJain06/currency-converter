# 🌍 Dynamic Currency Converter

A lightweight, modern web application that converts currencies in real-time using live exchange rates from a public API. 

## 🚀 Live Demo
You can check out the live version of the website here: **[Insert your GitHub Pages link here]**

## ✨ Features
* **Live Exchange Rates:** Fetches up-to-date conversion data automatically on page load.
* **Smart Defaults:** Pre-configures source as `USD` and target as `EUR` for a quick user experience.
* **Smart Math:** Seamlessly handles conversion between any two non-USD currencies using USD as a baseline.
* **Responsive UI:** Clean, modern glass-morphic card interface built from scratch.

## 🛠️ Built With
* **HTML5:** Semantic layout and structure.
* **CSS3:** Flexbox layouts, custom gradients, and interactive focus states.
* **JavaScript (ES6+):** Fetch API, `async/await` asynchronous logic, and dynamic DOM manipulation.
